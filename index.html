    Web 1.0 Web 2.0 Web 3.0

    2) How could I find a page? Only by specific URL Search engines retrieve tags with Micro content Live-streams / Waves = enables semantic search
    3) What WEB could do? Read-Only Read-Write Read, Write and execute web

    Question 2


    Query Example: "Side effects of Corona Medicine"

    • WEB 1.0: You will need to find the specific URL for this query.
    • WEB2.0: You can search the query: "Side effects of Corona Medicine" and you will get results that contains tags from the query words.
      Not always the results were good but there were fast.
    • WEB 3.0: If you will write "Side effects of Corona Medicine", you will get links to websites that contained this topic, because the search was by semantic idea.
      The search is taking more time but better results.
